2️⃣  Affiliate-network integrations (real & scalable)

Create a unified Affiliate Integration Service inside /services/affiliate.js (or affiliate_service.py on backend).

Network	What to use	How to connect safely

Amazon Associates	Product Advertising API 5.0	Each user enters their tracking ID; backend fetches product meta via Amazon’s API key stored in .env.
ShareASale	Merchant API	Use user’s affiliate ID + global token from your ShareASale developer account.
ClickBank	Marketplace API	Pull top products + hoplink builder per user ID.
CJ Affiliate (Commission Junction)	Open Developer API	OAuth token (your app-level). Users connect via account ID.
Impact Radius	Partner API	OAuth + campaign filter.


Backend endpoints

GET  /api/networks/list           → returns connected networks
POST /api/networks/connect        → user submits affiliate ID
GET  /api/products/import/<url>   → fetch product meta (auto-detect network)

These make your app truly “multi-network.”
Later you can add automation (cron) that syncs clicks/conversions nightly.

3️⃣  Social-network integration

Add a Social Share Service that creates and posts affiliate promo content.

Platform	Integration Type	Action

Facebook & Instagram	Meta Graph API	Share product post to user’s connected Page or IG Business account.
TikTok	TikTok Share Kit / Upload API	Generate caption + thumbnail → open share intent.
Twitter (X)	v2 API	Tweet product title + shortlink.
Pinterest	REST API	Pin image to user’s board.
YouTube Shorts	Upload API (planned)	Later allow uploading AI-generated ad videos.


🔐 All require OAuth consent screens. Each user connects their accounts once.
Store refresh tokens encrypted in DB.
