🔗 DATA FLOW SUMMARY

1️⃣ User registers → JWT created → redirected to dashboard.
2️⃣ Adds products → auto-linked to affiliate URLs.
3️⃣ Launches AdBoost → credits deducted → clicks recorded.
4️⃣ Commissions accumulate → user requests withdrawal.
5️⃣ Admin reviews/approves → funds released via Flutterwave.
6️⃣ Notifications update user in real time.
