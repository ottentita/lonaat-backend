D. WITHDRAWAL SYSTEM (FIXED & SECURE)
1. Withdrawal Eligibility
A user can request withdrawal ONLY if:
commission_status = approved
minimum withdrawal amount is reached
no fraud flags on account
2. Withdrawal Methods (CURRENT)
✅ Direct Bank Transfer only
Store:
bank_name
account_name
account_number
country
currency
(Stripe / Flutterwave placeholders remain disabled for now)
3. Withdrawal Flow
User clicks Withdraw
Enters amount (≤ available balance)
Confirms bank details
Request is created with status = pending
Admin reviews request
Admin approves → status = paid
System deducts balance
Withdrawal receipt logged
