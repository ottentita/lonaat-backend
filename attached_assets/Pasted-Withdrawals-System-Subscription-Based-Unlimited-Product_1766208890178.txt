Withdrawals System + Subscription-Based Unlimited Product Generation
This update introduces two core systems:
One-Click Withdrawals with Bank Details
Subscription-Controlled Unlimited Product Generation
Both apply to Admin and Users, with different permissions.
1️⃣ ONE-CLICK WITHDRAWALS SYSTEM (BANK TRANSFER)
🎯 Goal
Allow a user (and admin) to withdraw funds easily using saved bank details, with a simple and clear flow.
🔹 USER FLOW (FRONTEND)
Dashboard → Withdrawals
User clicks “Withdraw Funds”
System checks:
Available balance
Minimum withdrawal threshold (configurable)
If bank details are not saved:
Redirect to Bank Details Form
If bank details are already saved:
Show confirmation screen
User clicks “Confirm Withdrawal”
System submits withdrawal request
Status shown as:
Pending
Approved
Rejected
Paid
🔹 BANK DETAILS FORM (ONE-TIME OR EDITABLE)
Fields:
Bank Name
Account Holder Name
Account Number / IBAN
Country
Optional: SWIFT / Routing Code
Buttons:
Save Bank Details
Update Bank Details
⚠️ Bank details must be:
Encrypted at rest
Editable only by the account owner
Visible only to Admin (read-only)

🔹 ADMIN FLOW
Admin Dashboard → Withdrawals
Admin can:
View all withdrawal requests
Filter by:
User
Date
Status
Approve or reject withdrawals
Mark withdrawal as Paid
⚠️ Admin withdrawals
Admin can withdraw without limits
No subscription or credit check
Same bank details flow applies
