📧 6️⃣ EMAIL NOTIFICATION SERVICE

Configure SMTP (Brevo or Mailgun).

Environment vars:

EMAIL_HOST=smtp-relay.sendinblue.com
EMAIL_PORT=587
EMAIL_USER=your_email@domain.com
EMAIL_PASS=your_smtp_password
EMAIL_SENDER="Lonaat Support <no-reply@lonaat.com>"

Use email_service.py:

import smtplib
from email.mime.text import MIMEText
import os

def send_email(to, subject, message):
    msg = MIMEText(message, 'html')
    msg['Subject'] = subject
    msg['From'] = os.getenv("EMAIL_SENDER")
    msg['To'] = to
    with smtplib.SMTP(os.getenv("EMAIL_HOST"), int(os.getenv("EMAIL_PORT"))) as server:
        server.starttls()
        server.login(os.getenv("EMAIL_USER"), os.getenv("EMAIL_PASS"))
        server.send_message(msg)

Trigger on:

New registration (welcome)

Commission approval

Withdrawal paid

Fraud flag or security review


✅ Send test emails to three demo users.
