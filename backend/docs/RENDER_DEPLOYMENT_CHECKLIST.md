## 🔍 Troubleshooting Common Issues

### Issue 1: "Not Found" (404)

**Cause:** App is sleeping (free tier) or failed to start

**Fix:**
1. Wait 60 seconds and refresh
2. Check Render logs for errors
3. Verify environment variables are set

### Issue 2: "Firebase initialization failed"

**Cause:** Missing or invalid `FIREBASE_SERVICE_ACCOUNT`

**Fix:**
1. Verify JSON is complete (starts with `{` and ends with `}`)
2. No extra spaces or line breaks
3. Generate new service account key if needed

### Issue 3: Build Failed

**Cause:** Missing dependencies or wrong Python version

**Fix:**
1. Check `requirements.txt` includes all packages
2. Verify no syntax errors in Python files
3. Check Render logs for specific error

### Issue 4: "Module not found: main"

**Cause:** Incorrect start command

**Fix:**
- Verify start command is: `gunicorn --bind 0.0.0.0:$PORT --workers 2 --timeout 120 main:app`
- Your main file is `main.py` with Flask app named `app`
